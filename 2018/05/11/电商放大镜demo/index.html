电商放大镜demo

不久前做了一个电商网站的demo，其中自己写了个基于原生js的放大镜demo，
大致的需求是：鼠标放到小盒子上，让大盒子里面的图片和小盒子的图片同步等比例移动

html结构：

<div class="goods-info-message-l-l fl">
    <div id="small-Img">
        <img src="images/productInfo/nw11.JPG" id="goods-info-bigImg"/>
        <div class="goods-info-mask"></div>
    </div>
    <div id="magnifier">
        <img src="images/productInfo/nw11.JPG" alt=""/>
    </div>
</div>
<div class="goods-info-message-l-r fr">
    <ul id="goods-info-smallImg">
        <li class="goods-info-curren"><img src="images/productInfo/nw11.JPG" alt=""/></li>
        <li><img src="images/productInfo/nw12.JPG" alt=""/></li>
        <li><img src="images/productInfo/nw13.JPG" alt=""/></li>
        <li><img src="images/productInfo/nw14.JPG" alt=""/></li>
    </ul>
</div>

js代码，基于jq获取DOM节点：

1.获取相关元素：

var small = $("#small-Img")[0];
var bigImg = $("#magnifier>img")[0];
var maskly = $(".goods-info-mask")[0];
var box = $(".goods-info-message-l-l")[0];

2.绑定事件，鼠标放上去显示大盒子，移开隐藏大盒子。

small.onmouseenter = function () {
    $(".goods-info-mask").show();
    $("#magnifier").show();
}
small.onmouseleave = function () {
    $(".goods-info-mask").hide();
    $("#magnifier").hide();
}

3.实现功能，移动鼠标盒子，大盒子图片等比例移动

绑定的事件是onmousemove，而事件源是small(只要在小盒子上移动1像素，黄盒子也要跟随)

small.onmousemove = function (event) {
    //想移动鼠标盒子，必须知道鼠标在small中的位置。x作为maskly的left值，y作maskly的top值。
    event = event || window.event;
    var pagex = event.pageX || scroll().left + event.clientX;
    var pagey = event.pageY || scroll().top + event.clientY;

    //让鼠标在鼠标盒子最中间，减去鼠标盒子宽高的一半
    var x = pagex - box.offsetLeft - maskly.offsetWidth/2;
    var y = pagey - box.offsetTop - maskly.offsetHeight/2;

    //限制换盒子的范围,鼠标盒子不能超出外围大盒子
    //left取值为大于0，小盒子的宽-maskly的宽。
    if(x<0){
        x = 0;
    }
    if(x>small.offsetWidth-maskly.offsetWidth){
        x = small.offsetWidth-maskly.offsetWidth;
    }
    if(y<0){
        y = 0;
    }
    if(y>small.offsetHeight-maskly.offsetHeight){
        y = small.offsetHeight-maskly.offsetHeight;
    }
    //移动鼠标盒子
    maskly.style.left = x + "px";
    maskly.style.top = y + "px";

    //等比例移动大盒子图片，大图片我是等比例放大小图片
    //大图片走的距离/mask路由盒子都的距离 = 大图片/小图片
    var bili = bigImg.offsetWidth/small.offsetWidth;
    var xx = bili*x;
    var yy = bili*y;
    bigImg.style.marginTop = -yy+"px";
    bigImg.style.marginLeft = -xx+"px";
}

//最后写个图片切换的点击事件
$("#goods-info-smallImg").on("click","li", function () {
    $(this).addClass("goods-info-curren").siblings("li").removeClass("goods-info-curren");
    $("#goods-info-bigImg").attr("src",$(this).find("img").attr("src"));
    $("#magnifier img").attr("src",$(this).find("img").attr("src"));
})

总结：原理挺简单的，没事的时候谢谢小demo，练习下^_^
