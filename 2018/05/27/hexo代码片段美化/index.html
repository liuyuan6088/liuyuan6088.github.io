hexo 会有各种生命周期和事件，平时可能不会用到，但是能很好的利用的话，可以提高不少效率。比如文章多到一定程度之后，每次创建新文章都会被淹没在文件夹里面，在博客根目录下创建一个 scripts 文件夹，放一个 events.js 文件。这样每次通过hexo new post 创建新文章就会自动用 code 打开了~
